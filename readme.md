Project Overview: 

The Bahaghari Retail Co.  focuses on the data transformation and analytical lifecycle of a multi-regional retail chain in the Philippines, encompassing stores across Luzon, Visayas, and Mindanao. Faced with raw data challenges such as duplicate transactions, inconsistent categorization, and varying date formats, this project utilizes SQLite for robust data cleaning and standardization to ensure a reliable "single source of truth." By integrating structured store, product, and transaction datasets, the analysis provides actionable insights into sales performance, regional trends, and operational efficiencies, ultimately empowering the organization with a polished Power BI dashboard for executive-level decision-making.

Description:
- /01_statistics/: Contains the exploratory data analysis and statistical summaries in Excel.
- /02_data_preparation/: Includes a clean Excel workbook and  SQL scripts for data cleaning, transformation, and database normalization.
- /03_visualization/: Houses the Power BI files (.pbix) and related visual assets for reporting.
- /04_executive_summary/: Stores the final reports and executive summaries of the findings.
